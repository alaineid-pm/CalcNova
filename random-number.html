Random Number Generator — Cryptographically Secure Randomness

This generator uses crypto.getRandomValues() — the same cryptographic API used for security-critical applications. It draws on hardware entropy sources in your device, producing numbers that are statistically indistinguishable from true randomness. All modes (number range, dice, coin flip, list picker) use this same high-quality randomness source.

Number Range Generator

Generate one or multiple random integers in any range. Enable unique mode to ensure no duplicates — ideal for lottery draws, random sampling, or shuffling. The unique picker uses a Fisher-Yates shuffle for perfectly uniform selection without bias.

Dice Roller — d4 Through d100

Roll standard tabletop RPG dice: d4, d6, d8, d10, d12, d20, and d100 (percentile). Roll up to 10 dice at once and see individual results plus the total — perfect for D&D 5e, Pathfinder, Call of Cthulhu, or any TTRPG session. Average rolls: d6 = 3.5, d8 = 4.5, d12 = 6.5, d20 = 10.5.

Fair Coin Flip and List Picker

Flip up to 100 coins simultaneously and see heads/tails counts and percentages. The List Picker lets you paste any number of items and randomly select one or more winners — ideal for giveaways, team assignments, or decision-making. All selections are equally probable and unbiased.
